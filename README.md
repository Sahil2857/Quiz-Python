A simple command-line Quiz Bot built with Python.  
It asks the user multiple-choice questions, validates answers, and shows the final score.

## Features
- Multiple-choice questions
- Instant feedback (correct/incorrect)
- Score tracking
- Easy to add or edit questions
- Runs in any terminal/command prompt

## Requirements
- Python 3.x

No external libraries are required.

## How It Works
1. The bot displays a question with options (A, B, C, D).
2. The user types the correct option.
3. The bot checks the answer and updates the score.
4. After all questions, the final score is shown.

## How to Run

1. Download or clone this repository
2. Open a terminal in the project folder
3. Run:

```bash
python quiz_bot.py
